What this means
An AWS access key was found in a public JavaScript file. Anyone who finds this key can access your Amazon cloud account — they could read your databases, send emails on your behalf, or delete files.
How to fix
  1. Go to AWS Console → IAM → revoke this key immediately
  2. Move secrets to environment variables or a secrets manager
  3. Never commit API keys to source code or JavaScript files
