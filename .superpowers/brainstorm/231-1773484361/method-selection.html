A — Pill Toggles

All methods visible as clickable pills, grouped by section. Selected = colored, unselected = dim. See everything at once. Clean, fast, no nested menus. Beginner-friendly — each pill has a tooltip explaining what it does.

B — Category Cards (expand to configure)

Three cards: Passive / Recon / Active. Each shows a count badge and a plain-English description ("Safe analysis — no requests sent"). Click a card to open its checklist. Hides complexity by default but exposes full control on demand.

C — Plain-English Presets

Each scan mode is a big toggle row with a plain-English description of what it does and who should use it. Beginners see "Reads HTML & JS — finds secrets." Experts can still expand each row to tweak individual tests. Most beginner-friendly of the three.

These can be combined — e.g. Category Cards (B) with plain-English descriptions inside each card. Let me know what resonates.
