Findings Display — How should vulnerabilities be shown?

You want it informative for non-technical users. Here are two approaches to displaying scan results.

CRITICAL
AWS Secret Key Exposed
secrets
WHAT THIS MEANS
An AWS access key was found in a public JavaScript file. Anyone who finds this can access your Amazon cloud account — read databases, send emails, delete files.
HOW TO FIX
1. Revoke this key in AWS console immediately
2. Move secrets to environment variables
3. Never commit keys to source code
MEDIUM
Missing Security Headers
Your site doesn't set Content-Security-Policy headers. This makes it easier for attackers to inject malicious scripts into your pages. ▸ See fix

A — Explained Cards

Each finding has three sections: the name + severity badge, a plain-English "What this means" block, and a "How to fix" numbered list. Critical findings expanded by default, lower severity collapsed. Perfect for non-technical readers — no jargon.
